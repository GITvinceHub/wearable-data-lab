# Wearable Data Lab

## Overview

Wearable Data Lab is a technical repository focused on the analysis, comparison, and exploitation of data generated by wearable devices.

It investigates how ecosystems such as Garmin and Apple structure and expose activity data through formats like FIT, TCX, GPX, and Apple Health XML.

---

## Objectives

- Analyze the structure and semantics of activity file formats  
- Compare data richness across FIT, TCX, GPX, Apple XML  
- Quantify data loss during format conversion  
- Enable interoperability between closed and open ecosystems  
- Develop tools for parsing, comparison, and normalization  
- Provide reproducible workflows for wearable data analysis  

---

## Scope

This repository covers:

- Activity file formats  
  - FIT (binary, high fidelity)  
  - TCX (XML, sport-oriented)  
  - GPX (open GPS standard)  
  - Apple Health XML (global export)  

- Ecosystem interactions  
- Data pipelines and transformations  
- Reverse engineering and field-level analysis

## Key Topics

### 1. Format Comparison

- Field-level comparison between FIT, TCX, GPX, Apple XML  
- Structural differences (records, laps, sessions)  
- Metadata and sensor support  

### 2. Data Integrity

- Precision differences (GPS, speed, altitude)  
- Handling missing or derived data  

### 3. Interoperability

- Apple Health → standard formats via tools
